* アイコンをクリックするとデレる。

- 問い合わせ結果を Slack へ通知
- チャット風のデザインとアニメーション
- 使用言語
- JavaScript(React), JSX

# 技術

- Sass, React, Material-ui, Firebase Hosting, Firestore, Cloud,Functions
